# HNS Mix System

# Как работает?

Запуск микса
Админ пишет /mode
Выбирает "Mix"
Выбирает тип: Boost или Skill

Boost режим:
Semiclip выключен (можно бустить)
Freezetime из настроек MS_FREEZEBOOST
Выдаётся Helper каждой команде

Skill режим:
Semiclip включен (проходишь сквозь тиммейтов)
Freezetime из настроек MS_FREEZESKILL
Команды TT (Terrorists)
При спавне получают:

Нож
Флешки (количество из MS_FLASH_TT)
Смоки (количество из MS_SMOKE_TT)
Бесшумные шаги
Команды CT
При спавне получают:

Нож
HP из настроек MS_HP
Обычные шаги (слышно)
Роли (/roles, /role, /use)
Игрок выбирает роль в начале микса. Каждая роль одноразовая.

Роль	Команда	Способность
Helper	TT/CT	/rr — рестарт раунда (только в начале)
Lively	TT/CT	+50 HP, +100 Armor
Grenader	TT	+1 Smoke, +1 Flash
Survivor	CT	Респавн после смерти
Smoker	TT	+2 Smoke
Flasher	TT	+2 Flash
Tank	TT/CT	+35 HP, +255 Armor
Victim	TT	Отдаёт своё HP союзнику
Scout	TT/CT	Получает Scout (без патронов)
Hider	CT	Бесшумные шаги
Healer	TT/CT	+15 HP всей команде
Использование: /use или через меню /roles

Ограничения:
Способность активируется через 15 сек после freezetime
Helper может рестартить только в первые секунды раунда
Одна роль = один игрок в команде
Замена игроков (/re, /replace)
Игрок в команде может позвать спектатора заменить себя:

Пишет /re
Выбирает спектатора из списка
Спектатор получает меню "Accept?"
Если принял — занимает место, получает позицию, HP, гранаты, роль
Блокировки:

Нельзя заменять когда до победы < 3 минут
Нельзя звать игроков с /noplay или забаненных
Контроль (/co, /control, G)
Живой игрок отдаёт управление тиммейту:

Пишет /co или нажимает G
Выбирает тиммейта
Тиммейт получает приглашение
Если принял — получает позицию и продолжает играть
Лимит: 3 контроля за микс на игрока

Noplay (/noplay)
Спектатор пишет /noplay:

Его не будут приглашать на замену
Повторно /noplay — отключает
Баны (/ban, /unban, /banlist)
Админ может забанить игрока:

Mix Ban:

Не может играть в Mix/Knife/Captain
Кидает в спектаторы
Full Ban:

Только спектатор
Во всех режимах
AFK система
Во время микса:

После freezetime проверяет позиции
Если игрок не двигался — пауза
HUD показывает AFK игроков
Когда все вернулись — автоматический unpause
Остановка микса
Админ через /mode выбирает "Training":

Freezetime сбрасывается на 2 сек
Semiclip выключается
Все роли сбрасываются

Дополнительные команды:

Команда	Описание
/boost	Админ переключает на Boost
/skill	Админ переключает на Skill
/mode	Меню режимов
/roles	Список ролей игроков
/role	Показать свою роль
